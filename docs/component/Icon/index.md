# Icon 图标

**Elin-UI** 为您提供了一套常用的图标。

## 使用方法

每个图标都是一个vue组件，可直接导入使用，统一以【Iconxxx】命名。

```vue
<template>
  <IconPlus/>
</template>

<script setup lang="ts">
  import {IconPlus} from 'elin-ui'
</script>
```
